SRP project summary

This is a webserver presenting the secondary data analysis output of the RNA-seq data used in the Nature Biotechnlogy publication by Wang et al. 2012. We used three RNA-seq reads from the chemical PIR and a set of three controls.

The Tuxedo software suite by Trapnell and others comprising of Bowtie, Tophat, Cufflinks and the R package CummeRbund were used in data analysis and results presentation.
